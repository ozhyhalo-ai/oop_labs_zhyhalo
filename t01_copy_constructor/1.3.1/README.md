# Аналізатор геометричних фігур

### Що робить програма
Ця програма на Python читає дані про геометричні фігури (трикутник, прямокутник, трапеція, паралелограм, круг) із текстових файлів.
Вона використовує принципи ООП для створення об'єктів цих фігур.

Програма автоматично перевіряє правильнисть введених даних (наприклад, чи виконується нерівність трикутника та чи не дорівнюють сторони нулю).
Після цього вона обчислює площу та периметр кожної правильної фігури і знаходить ті, що іають найбільші значення.

### Як запустити
1. Переконайтеся, що у вас встановлений Python 3.
2. Завантажте файл з кодом ('l_01.py') та текстові файли з даними ('input01.txt', 'input02.txt', 'input03.txt') в одну папку.
3. Відкрийте термінал, перейдіть у цю папку та виконайте команду:  
'python l_01.py'

### Приклад вводу/виводу

**Приклад вводу (з файлу input01.txt):**  
Rectangle 22 5  
Parallelogram 0 8 21  
Triangle 14 16 0  
Circle 25

**Приклад виводу в консоль:**  
Обробка файла input01.txt  
У список додано 12 правильних фігур  
Найбільша площа: Circle(r=25.0) (Площа: 1963.50)  
Найбільший периметр: Circle(r=25.0) (Периметр: 157.08)
