# Генератор букета квітів (Turtle Graphics)

### Що робить програма
Ця програма на Python використовує модуль `turtle` для створення візуальної сцени - малювання яскравого букета квітів на екрані.

Програма сурово дотримується принципів об'єктивно-орієнтованого програмування і демонструє використвння **композиції**.
Головний клас `Bouquet` є контейнером для списку унікальних об'єктів `Flower` (Квітка). Кожна квітка, у свою чергу, самостійно
створює та збирає свої складові частини за допомогою класів: `Stem` (Стебло), `Petal` (Пелюстка) та `Leaf` (Листок).

Алгоритм автоматично вираховує координати для переплетіння стебел у спільний вузол букета, малює пелюстки у формі крапель
та динамічно додає великі листки у формі дельтоїда виключно до високих квітів.

### Як запустити
1. Переконайтеся, що у вас встановлений Python 3. 
2. Завантажте файл з кодом (`l_02.py`) у будь-яку зручну папку.
3. Відкрийте термінал, перейдіть у цю папку та виконаєте команду:  
`python l_02.py`

### Приклад вводу/виводу

Оскільки це графічна програма, введення параметрів відбувається під час ініціалізації об'єктів у коді, а виводом є візуалізація
на екрані, а не текст у консолі.

**Приклад вводу (фрагмент коду з l_02.py):**  
`Flower(top_x=0, top_y=130, bottom_x=0, petal_color="magenta")`

**Приклад виводу (графічне вікно Turtle):**  
Програма відкриває вікно і малює по центру екрана квітку кольору маджента. До неї знизу підводиться зелене стебло. Оскільки 
висота цієї квітки (130) не перевищує поріг (150), програма автоматично розуміє, що дельтоїдний листок малювати не потрібно. 
Загалом генерується букет із 5 таких унікальних квіток, після чого курсор-черепашка ховається, залишаючи готовий малюнок.
