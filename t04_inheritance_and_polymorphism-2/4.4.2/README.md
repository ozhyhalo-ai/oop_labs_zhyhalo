# Гра "Хрестики-нулики" (Наслідування та поліморфізм)

### Що робить програма
Ця програма на Python реалізує класичну гру "Хрестики-нулики" з графічним інтерфейсом за допомогою бібліотеки `turtle`.

Програма побудована на принципах об'єктно-орієнтованого програмування, зокрема **наслідування** та **поліморфізму**. 
В основі лежить базовий абстрактний клас `Figure`, який відповідає за збереження координат та загальні дії (поява `show()` 
та зникнення `hide()`). Від нього наслідуються конкретні класи: ігрове поле (`Board`), хрестик (`Cross`) та нулик (`Nought`).

Завдяки поліморфізму, кожен з класів-нащадків має власну реалізацію віртуального методу `draw(color)`, що дозволяє їм 
малювати свої унікальні форми. Логіка гри використовує внутрішню матрицю пам'яті (список списків 3x3) для відслідковування
вільних клітинок, чергування ходів гравців та автоматичної перевірки перемоги (по горизонталі, вертикалі чи діагоналі) 
або нічиї.

### Як запустити
1. Переконайтеся, що у вас встановлений Python 3.
2. Завантажте файл з кодом (`l_04.py`) у будь-яку зручну папку.
3. Відкрийте термінал, перейдіть у цю папку та виконайте команду:  
`python l_04.py`

### Приклад вводу/виводу

Оскільки це інтерактивна графічна програма, керування відбувається за допомогою миші.

**Приклад вводу:**  
Користувач клікає лівою кнопкою миші по порожніх клітинках ігрового поля розміром 3x3.

**Приклад виводу:**  
Програма автоматично вираховує центр обраної клітинки та малює там відповідну фігуру (чергуючи хрестики та нулики). Якщо 
гравець збирає три фігури в ряд, програма перекреслює їх червоною лінією та виводить по центру екрана вікно з написом на 
рожевому фоні, наприклад: `Перемога: X!`. Якщо всі клітинки заповнені, а переможця немає, з'являється сірий банер `Нічия!`.
